O que é linguagem HTML? 

* <img src="" alt=""> : imagem + link da imagem + discrição da imagem
* <h1></h1> : Título
* <p></p> : Parágrafo/texto
* <input type="text" + placeholder/> : abrir uma caixa de pergunta + escrito dentro da caixa
* <button></button> : botão com algo escrito dentro
